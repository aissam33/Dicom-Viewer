Mode d’emploi
Ce projet est une application web autonome (standalone), contenue dans un seul fichier HTML, qui intègre le code HTML, CSS et JavaScript. Elle peut être utilisée localement, sans serveur externe ni installation.
Instructions d’utilisation

Étape 1 : Télécharger l’application

1. Récupérez le fichier ‘index.html’ fourni avec ce projet.
2. Placez-le dans un dossier de votre choix sur votre ordinateur.

Étape 2 : Lancer l’application

1. Double-cliquez sur le fichier ‘index.html’.
2. Il s’ouvrira automatiquement dans votre navigateur par défaut.

Remarque :Si cela ne fonctionne pas, cliquez droit sur le fichier > Ouvrir avec> sélectionnez votre navigateur.

Étape 3 : Utilisation de l’interface

1. Suivez les instructions à l’écran (par exemple : charger une image, cliquer sur un bouton, etc.).
2. L’application s’exécute entièrement dans votre navigateur — aucune connexion Internet n’est requise.

Personnalisation (facultatif)

Vous pouvez modifier le contenu, le style ou le comportement de l’application en ouvrant le fichier index.html dans un éditeur de texte ou de code comme :
- Visual Studio Code](https://code.visualstudio.com)
- Notepad++
- Sublime Text
